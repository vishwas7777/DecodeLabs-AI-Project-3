## Project Title
AI Recommendation System

## Developed By
Vishwas Parashar

## Description
This project recommends suitable career roles based on user skills using TF-IDF Vectorization and Cosine Similarity.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Workflow
- Dataset Creation
- TF-IDF Vectorization
- User Skill Input
- Cosine Similarity
- Top 3 Recommendations
- Visualization

## Output
Top Recommended Roles based on user skills.
